# Раздел 1: Введение

**Хронометраж:** 1-2 минуты

---

## Текст для озвучки

Добро пожаловать в систему управления знаниями на основе искусственного интеллекта.

Эта система создана, чтобы помочь вам быстро находить информацию в больших массивах документов, получать ответы на вопросы по судебной практике и автоматизировать рутинные задачи.

Одна из ключевых возможностей системы — транскрибация аудиозаписей. Вы можете загружать аудиопротоколы судебных заседаний, и система автоматически преобразует их в текст. После этого доступны инструменты для обработки стенограмм: форматирование, выделение ключевых моментов, создание резюме.

В этом видео мы пройдём полный путь от регистрации до практического использования системы. Вы научитесь создавать базы знаний, загружать документы судебной практики, настраивать искусственный интеллект для поиска по вашим документам, а также работать с транскрибацией и обработкой аудиозаписей.

Давайте начнём с регистрации в системе.
