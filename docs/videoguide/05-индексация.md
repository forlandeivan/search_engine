# Раздел 5: Индексация базы знаний

**Хронометраж:** 3 минуты

---

## Текст для озвучки

Индексация — это процесс преобразования текста документов в специальный формат, который позволяет искусственному интеллекту быстро находить релевантную информацию.

Без индексации система не сможет отвечать на вопросы по вашим документам. Это обязательный шаг после загрузки документов.

В панели базы знаний найдите кнопку "Индексировать". Нажмите на неё, чтобы открыть мастер индексации.

Система предлагает два режима.

Экспресс-индексация — быстрый запуск с настройками по умолчанию. Рекомендуется для первого знакомства с системой.

Расширенная настройка — позволяет выбрать модель эмбеддингов, размер фрагментов текста и другие параметры. Используйте этот режим, если вам нужна тонкая настройка.

Выбираем экспресс-индексацию и нажимаем "Запустить индексацию".

Начинается процесс обработки документов. Вы видите прогресс — сколько документов обработано из общего количества.

Время индексации зависит от количества и объёма документов. Для базы в несколько сотен документов это обычно занимает от одной до пяти минут.

По завершении статус базы знаний изменится на "Индексирована". Это означает, что база готова к использованию в поиске.

Обратите внимание: при добавлении новых документов потребуется повторная индексация. Система сама определит, какие документы изменились, и проиндексирует только их.

Теперь создадим навык, который будет использовать нашу базу знаний для ответов на вопросы.

---

## Что показывать на экране

1. База знаний с загруженными документами
2. Кнопка "Индексировать" — наведение и клик
3. Мастер индексации (IndexingWizardModal):
   - Выбор режима: "Экспресс" vs "Расширенная"
   - Выбор "Экспресс-индексация"
4. Нажатие "Запустить индексацию"
5. Прогресс индексации:
   - Индикатор прогресса
   - Счётчик обработанных документов
6. Завершение индексации
7. Статус базы "Индексирована" (зелёный индикатор)
