# Раздел 9: Действия для обработки стенограмм

**Хронометраж:** 3-4 минуты

---

## Текст для озвучки

Действия — это инструменты на основе искусственного интеллекта для обработки текста. Они позволяют автоматизировать рутинные задачи: форматирование, резюмирование, выделение ключевых моментов.

Чтобы создать действие, перейдите в соответствующий раздел в боковом меню.

Нажмите кнопку создания и заполните поля.

В поле названия укажите понятное имя. Например, "Краткое содержание".

В поле описания кратко опишите, что делает это действие.

Параметр "Цель" определяет, к чему применяется действие. Для работы со стенограммами выберите "Стенограмма".

Параметр "Площадка" определяет, где будет доступно действие. Выберите "Canvas", чтобы действие появилось в холсте стенограммы.

Параметр "Режим вывода" определяет, как отображается результат. "Документ" создаёт отдельную вкладку с обработанным текстом, сохраняя оригинал. Замена - меняет содержимое исходника.

Самая важная часть — промпт. Это инструкция для искусственного интеллекта. Опишите, что должно делать действие. Например: "Сделай краткое содержание стенограммы, выдели ключевые моменты. Вот исходный материал - и здесь пишем слово текст в фигурных скобках."

Сохраняем действие.

Затем активируем его в нашем навыке.

Теперь вернёмся к холсту со стенограммой. В меню действий появился новый пункт "Краткое содержание". Нажмите на него.

Искусственный интеллект обрабатывает текст согласно вашим инструкциям. Результат появляется в новой вкладке. Оригинальная стенограмма сохраняется.

Вы можете создать несколько действий для разных задач. Например, действие для выделения показаний свидетелей, действие для создания краткого резюме заседания, действие для извлечения дат и сроков.

Каждое действие — это инструмент, который вы настраиваете под свои потребности и используете многократно.

---

## Что показывать на экране

1. Переход в раздел "Действия" (WorkspaceActionsPage)
2. Список существующих действий (если есть)
3. Нажатие кнопки создания нового действия
4. Форма создания действия:
   - Поле "Название" — ввод "Форматирование протокола"
   - Поле "Описание" — краткий текст
   - Параметр "Цель" (target):
     - Показать опции: Стенограмма, Сообщение, Выделение, Диалог
     - Выбор "Стенограмма"
   - Параметр "Площадка" (placement):
     - Чекбоксы: Canvas, Message, Toolbar
     - Выбор "Canvas"
   - Параметр "Режим вывода" (outputMode):
     - Показать опции
     - Выбор "Новая версия"
   - Поле "Промпт" — ввод инструкции:
     "Отформатируй текст протокола судебного заседания. Раздели на абзацы по выступлениям участников. Исправь очевидные ошибки распознавания речи. Сохрани юридическую терминологию."
5. Сохранение действия
6. Действие появляется в списке
7. Переход к холсту со стенограммой (TranscriptCanvas)
8. Меню действий — показать новый пункт "Форматирование протокола"
9. Клик на действие
10. Индикатор обработки
11. Результат в новой вкладке:
    - Вкладка "Оригинал"
    - Вкладка "Форматирование протокола" (результат)
    - Отформатированный текст
12. Упоминание других возможных действий (можно показать список примеров)
