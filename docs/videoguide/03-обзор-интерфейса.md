# Раздел 3: Обзор интерфейса

**Хронометраж:** 2 минуты

---

## Текст для озвучки

После входа вы попадаете в основной интерфейс системы. Слева расположена панель навигации с основными разделами.

Раздел "Базы знаний" — это хранилище ваших документов. Здесь вы создаёте базы, загружаете документы и настраиваете индексацию для поиска.

Раздел "Навыки" — это настройка искусственного интеллекта. Навык определяет, как система будет отвечать на ваши вопросы, какие документы использовать и какие дополнительные функции применять.

Раздел "Действия" — это набор инструментов для обработки текста. Например, действия для форматирования стенограмм, выделения ключевых моментов или создания резюме.

Раздел "Чат" — это интерфейс общения с искусственным интеллектом. Здесь вы задаёте вопросы и получаете ответы на основе ваших документов.

В верхней части панели находится переключатель рабочих пространств. Рабочее пространство — это изолированная среда для работы. Вы можете создавать разные пространства для разных проектов или категорий дел.

Теперь перейдём к созданию базы знаний.

## Что показывать на экране

- Боковая панель навигации (MainSidebar)
- Поочерёдное выделение разделов:
  - "Базы знаний" — наведение/клик
  - "Навыки" — наведение/клик
  - "Действия" — наведение/клик
  - "Чат" — наведение/клик
- Переключатель рабочих пространств (WorkspaceSwitcher)
- Клик на переключатель, показать выпадающий список
