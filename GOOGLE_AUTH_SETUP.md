# Настройка входа через Google

1. В консоли Google Cloud создайте OAuth 2.0 Client ID для веб-приложения. В разделах **Authorized JavaScript origins** и **Authorized redirect URIs** добавьте адрес панели и Callback URL из формы (по умолчанию `/api/auth/google/callback`).
2. В административной панели откройте раздел **Настройки → Настройки входа**. Введите `Client ID`, `Client Secret` и при необходимости измените `Callback URL`.
3. Включите переключатель «Включить вход через Google» и сохраните изменения. Настройки применяются сразу; при успешной конфигурации пользователи смогут авторизоваться через корпоративные аккаунты Google.

> ⚠️ Если ранее использовались переменные окружения `GOOGLE_CLIENT_ID` и `GOOGLE_CLIENT_SECRET`, сохранение формы создаст запись в базе данных и переопределит параметры окружения.
